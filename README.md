# Quick.ai — AI SaaS Platform

A full-stack AI SaaS application with subscription billing.

🔗 **Live Demo:** https://quick-ai-rho-pink.vercel.app
💻 **GitHub:** https://github.com/rishabh1606bits/quick-ai

## Features
- ✍️ Article Generator — Generate full articles from a title
- 📝 Blog Title Generator — SEO-friendly titles by keyword
- 🎨 Image Generator — AI images from text prompts
- 🖼️ Background Remover — Remove image backgrounds instantly
- 📄 Resume Analyzer — Complete AI feedback on resumes
- 💳 Subscription Billing — Free & Premium plans via Razorpay
- 🔐 Authentication — Secure sign-in via Clerk

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Clerk Auth
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Neon Serverless)
- **AI APIs:** Groq (Llama 3.3), Pollinations.ai, Remove.bg
- **Billing:** Razorpay Subscriptions
- **Deployment:** Vercel
